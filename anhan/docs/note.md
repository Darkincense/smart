 - mode ="aspectFit"
保持纵横比缩放图片，使图片的长边能完全显示出来

- video,image与view上下有间隙， dispaly:block
- `"navigationStyle": "custom"` 自定义小程序导航栏
- `<view> \n</view>`文本换行
