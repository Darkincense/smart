### Git commit log
* Head
  * type: feat 新特性, fix 修改问题, docs 文档, style 格式, refactor 重构, test 测试用例, chore 其他修改, 比如构建流程, 依赖管理.
  * scope:影响范围， 比如: route, component, utils, build... 可省略
  * subject:简短的提交信息
* Body
  * what：详细做了什么
  * why： 为什么这样做
  * how： 有什么后果
* Footer
  * 相关链接
